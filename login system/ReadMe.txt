Installation Instructtion:
==================================================
1. Import the "users.sql" file into the database.
2. Move all files to the same directory of your CodeIgniter application.
2. Move the "assets/" directories to the project's root folder.
3. Open the URL (http://localhost/project_folder_name/registration/) on the browser and check the registration and login functionality.
